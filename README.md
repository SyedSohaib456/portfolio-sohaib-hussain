This portfolio's visual styling and animation framework is a derivative work **inspired by and evolved from [Austin Serb's Portfolio](https://github.com/Austin1serb/portfolio-austin-serb)**, which was in turn inspired by Joseph Alexander’s Framer concept, *Launchfolio*.

## 🛠️ The Tech Stack & Architecture

This portfolio serves as a playground for production-level frontend engineering:

*   **Framework:** Next.js 14+ (App Router, React Server Components)
*   **Runtime & Package Manager:** Bun (extremely fast task execution)
*   **Styling:** Tailwind CSS + custom HSL color tokens for smooth dark/light mode balance
*   **State & Performance:** Integrates **React Zero UI** for zero-re-render animation management under heavy interactive scroll frames
*   **Animations:** Framer Motion + CSS keyframes tailored for buttery-smooth fluid rendering

---

## ✨ Unique Features & Customizations

1.  **3D Mechanical Keyboard Hover Effect:** Custom-engineered hover states in the Services grid that collapse the card's 3D bottom shadow and push it down by `4px` with a mechanical cubic-bezier timing curve.
2.  **Accelerated Scroll Snap:** Highly responsive hero card snap offsets (`8%` desktop / `6%` mobile) that slide the project mockups cleanly into their grid cells without interfering with read copy.
3.  **Custom Handwriting Signature:** Custom, transparent ink signature for **Sohaib Hussain** integrated into the balanced two-column About Section layout.
4.  **Specializations Outlines:** Standardized custom SVG mobile graphics that inherit `currentColor` dynamically without color inversion bugs.
